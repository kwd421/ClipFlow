# ClipFlow 1.0.5

## 변경 사항

- 시작할 때 업데이트 확인 때문에 잠깐 **응답 없음**이 뜨던 문제를 수정했습니다. 이제 백그라운드에서 확인하고 토스트만 띄웁니다.
- 구간선택 팝업에서 **정확 컷 / 빠른 컷** 선택 UI를 제거했습니다. 구간 다운로드는 항상 빠른 컷으로 동작합니다.
- URL 입력창 포커스 시 파란 테두리가 보이지 않도록 수정했습니다.
- 검색창 한글 IME 조합 중에도 필터가 바로 따라오도록 수정했습니다.
- 카드·썸네일 커서는 화살표, 링크·아이콘만 손가락 포인터로 맞췄습니다.

## 업데이트 안내

1.0.4에서 실행하면 앱 시작 후 잠시 뒤 업데이트 알림이 뜹니다. **1.0.3은 자동 업데이트 DLL 문제로 알림이 나오지 않을 수 있으니** 1.0.4 이상에서 확인해 주세요.
