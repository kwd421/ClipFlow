# ClipFlow 1.0.9

## 변경 사항

### 찌꺼기 정리
- 앱 시작 시 `%TEMP%`의 `clipflow-*` 임시 폴더·파일을 **1시간 지난 것만** 정리합니다. (이전 6시간 → 1시간)
- 저장 폴더의 **고아 partial** 청소를 추가했습니다.
  - 대상: `.part`, `.part.media`, `.trim.part.*`, `.part-Frag*` 등 이어받기/조각 파일만
  - **완성 `.mp4` 등은 삭제하지 않습니다**
  - 지금 목록에 있는 카드·받는 중인 경로의 partial은 **보호**합니다
  - **1시간 이상 지난** 파일만 지워서, 방금 받는 중인 파일과 충돌하지 않게 했습니다
- 크래시 후 저장 폴더에만 남은 `.part`도, 다음에 앱을 켜면(1시간 경과 시) 치워집니다.

### 업데이트
- 업데이트 설치 직후 **이전 창이 멈추던 문제**를 고쳤습니다. (WinSparkle 종료를 메인 스레드에서 처리, 종료 대기 단축)
- 패키지 빌드에서는 창 제목에 **버전 번호**가 표시됩니다. (예: `ClipFlow 1.0.9`)

### 참고
- 1.0.8 자동 업데이트 서명 오류가 난 경우, 이 버전을 **수동 설치**한 뒤부터는 이후 업데이트가 다시 맞춰집니다. (Windows 업데이트 키는 맥 Sparkle 키와 별개입니다.)
